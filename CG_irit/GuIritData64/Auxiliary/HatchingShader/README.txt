This folders contains the textures that are used in the hatching shader.

The images have to be toroidal (i.e. periodic), each image represents
a hatching tone or brightness, different patterns can be used to
achieve different results.

Based on a technique by Prain .E et al.

Default hatching assets are by Jaume Sanchez Elias
